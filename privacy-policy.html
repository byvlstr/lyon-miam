Politique de confidentialité

Dernière mise à jour : juin 2026
Cette politique de confidentialité décrit comment Lyon Miam collecte, utilise et protège vos informations lorsque vous utilisez notre application web accessible sur ce site.

1. Données collectées

Lyon Miam est une application web statique. Nous ne collectons pas directement de données personnelles identifiables (nom, email, téléphone, etc.).

Les seules données traitées lors de votre visite sont :

  • Données de navigation anonymes — collectées via Google AdSense (voir section 3)
  • Requêtes vers l'API Overpass / OpenStreetMap — votre adresse IP peut être transmise aux serveurs Overpass lors du chargement des données de restaurants

2. Données OpenStreetMap

Les données de restaurants affichées dans l'application proviennent d'OpenStreetMap, une base de données géographique libre et collaborative, sous licence ODbL.

Lors du chargement de l'application, une requête est envoyée à l'API Overpass (overpass-api.de). Cette requête peut transmettre votre adresse IP au serveur Overpass. Nous n'avons aucun contrôle sur les données collectées par ces serveurs tiers.

Les données OpenStreetMap peuvent être incomplètes ou incorrectes. Vous pouvez contribuer à leur amélioration sur openstreetmap.org.

3. Google AdSense et cookies publicitaires

Ce site utilise Google AdSense pour afficher des publicités. Google AdSense utilise des cookies afin de personnaliser les annonces en fonction de vos visites sur ce site et d'autres sites web.

Google peut utiliser les données collectées pour :

  • Afficher des publicités personnalisées basées sur vos intérêts
  • Mesurer l'efficacité des annonces
  • Éviter de vous montrer les mêmes annonces trop souvent

Vous pouvez désactiver la personnalisation des annonces Google en visitant google.com/settings/ads ou en installant le module de désactivation Google Analytics.

Pour en savoir plus sur la politique de confidentialité de Google : policies.google.com/privacy

4. Cookies

Lyon Miam n'utilise pas de cookies propres. Les cookies présents sur ce site proviennent exclusivement de Google AdSense (cookies tiers à des fins publicitaires).

Vous pouvez gérer ou désactiver les cookies via les paramètres de votre navigateur. Notez que la désactivation des cookies peut affecter le fonctionnement des publicités mais n'affecte pas les fonctionnalités principales de l'application.

5. Données de tiers

L'application charge des ressources depuis les services tiers suivants :

  • Google Fonts — polices de caractères (politique de confidentialité)
  • Leaflet / CartoDB — tuiles cartographiques pour la carte interactive (politique de confidentialité CARTO)
  • jsDelivr — bibliothèque JavaScript (topojson) hébergée sur CDN
  • Overpass API / OpenStreetMap — données de restaurants en temps réel

Chacun de ces services peut collecter des données de navigation (adresse IP, user-agent) conformément à leurs propres politiques de confidentialité.

6. Droits des utilisateurs (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès à vos données personnelles
  • Droit de rectification
  • Droit à l'effacement ("droit à l'oubli")
  • Droit d'opposition au traitement
  • Droit à la portabilité des données

Pour exercer ces droits, vous pouvez nous contacter. Concernant les données traitées par Google AdSense ou Overpass API, veuillez vous référer directement à leurs politiques respectives.

7. Sécurité

Lyon Miam est une application web statique sans base de données propre et sans formulaire de collecte de données. Aucune donnée personnelle n'est stockée sur nos serveurs.

8. Modifications

Cette politique de confidentialité peut être mise à jour à tout moment. La date de dernière mise à jour est indiquée en haut de cette page. Nous vous encourageons à la consulter régulièrement.

9. Contact

Pour toute question relative à cette politique de confidentialité ou à l'utilisation de vos données, vous pouvez nous contacter via l'adresse contact de notre site web (contact@lyon-Miam.org).
